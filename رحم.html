اسم فاعل
وَأَنتَ أَرْحَمُ الرَّاحِمِينَ ٧ : ١٥١ : ١١
وَهُوَ أَرْحَمُ الرَّاحِمِينَ ١٢ : ٦٤ : ١٧
وَهُوَ أَرْحَمُ الرَّاحِمِينَ ١٢ : ٩٢ : ١١
وَأَيُّوبَ إِذْ نَادَىٰ رَبَّهُ أَنِّي مَسَّنِيَ الضُّرُّ وَأَنتَ أَرْحَمُ الرَّاحِمِينَ ٢١ : ٨٣ : ١٠
إِنَّهُ كَانَ فَرِيقٌ مِّنْ عِبَادِي يَقُولُونَ رَبَّنَا آمَنَّا فَاغْفِرْ لَنَا وَارْحَمْنَا وَأَنتَ خَيْرُ الرَّاحِمِينَ ٢٣ : ١٠٩ : ١٤
وَقُل رَّبِّ اغْفِرْ وَارْحَمْ وَأَنتَ خَيْرُ الرَّاحِمِينَ ٢٣ : ١١٨ : ٧

اسم بصيغة «أَرْحام»

وَلَا يَحِلُّ لَهُنَّ أَن يَكْتُمْنَ مَا خَلَقَ اللَّهُ فِي أَرْحَامِهِنَّ إِن كُنَّ يُؤْمِنَّ بِاللَّهِ وَالْيَوْمِ الْآخِرِ ٢ : ٢٢٨ : ١٥
هُوَ الَّذِي يُصَوِّرُكُمْ فِي الْأَرْحَامِ كَيْفَ يَشَاءُ ٣ : ٦ : ٥
وَاتَّقُوا اللَّهَ الَّذِي تَسَاءَلُونَ بِهِ وَالْأَرْحَامَ ٤ : ١ : ٢٣
قُلْ آلذَّكَرَيْنِ حَرَّمَ أَمِ الْأُنثَيَيْنِ أَمَّا اشْتَمَلَتْ عَلَيْهِ أَرْحَامُ الْأُنثَيَيْنِ ٦ : ١٤٣ : ١٧
قُلْ آلذَّكَرَيْنِ حَرَّمَ أَمِ الْأُنثَيَيْنِ أَمَّا اشْتَمَلَتْ عَلَيْهِ أَرْحَامُ الْأُنثَيَيْنِ ٦ : ١٤٤ : ١٥
وَأُولُو الْأَرْحَامِ بَعْضُهُمْ أَوْلَىٰ بِبَعْضٍ فِي كِتَابِ اللَّهِ ٨ : ٧٥ : ١١
اللَّهُ يَعْلَمُ مَا تَحْمِلُ كُلُّ أُنثَىٰ وَمَا تَغِيضُ الْأَرْحَامُ وَمَا تَزْدَادُ ١٣ : ٨ : ٩
وَنُقِرُّ فِي الْأَرْحَامِ مَا نَشَاءُ إِلَىٰ أَجَلٍ مُّسَمًّى ثُمَّ نُخْرِجُكُمْ طِفْلًا ثُمَّ لِتَبْلُغُوا أَشُدَّكُمْ ٢٢ : ٥ : ٢٩
إِنَّ اللَّهَ عِندَهُ عِلْمُ السَّاعَةِ وَيُنَزِّلُ الْغَيْثَ وَيَعْلَمُ مَا فِي الْأَرْحَامِ ٣١ : ٣٤ : ١١
وَأُولُو الْأَرْحَامِ بَعْضُهُمْ أَوْلَىٰ بِبَعْضٍ فِي كِتَابِ اللَّهِ مِنَ الْمُؤْمِنِينَ وَالْمُهَاجِرِينَ إِلَّا أَن تَفْعَلُوا إِلَىٰ أَوْلِيَائِكُم مَّعْرُوفًا ٣٣ : ٦ : ٩
فَهَلْ عَسَيْتُمْ إِن تَوَلَّيْتُمْ أَن تُفْسِدُوا فِي الْأَرْضِ وَتُقَطِّعُوا أَرْحَامَكُمْ ٤٧ : ٢٢ : ١٠
لَن تَنفَعَكُمْ أَرْحَامُكُمْ وَلَا أَوْلَادُكُمْ ٦٠ : ٣ : ٣

اسم بصيغة «أَرْحَم»

وَهُوَ أَرْحَمُ الرَّاحِمِينَ ١٢ : ٦٤ : ١٦
وَهُوَ أَرْحَمُ الرَّاحِمِينَ ١٢ : ٩٢ : ١٠
وَأَيُّوبَ إِذْ نَادَىٰ رَبَّهُ أَنِّي مَسَّنِيَ الضُّرُّ وَأَنتَ أَرْحَمُ الرَّاحِمِينَ ٢١ : ٨٣ : ٩

اسم بصيغة «رَحِيم»

بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ ١ : ١ : ٤
الرَّحْمَٰنِ الرَّحِيمِ ١ : ٣ : ٢
إِنَّهُ هُوَ التَّوَّابُ الرَّحِيمُ ٢ : ٣٧ : ١١
إِنَّهُ هُوَ التَّوَّابُ الرَّحِيمُ ٢ : ٥٤ : ٢٦
إِنَّكَ أَنتَ التَّوَّابُ الرَّحِيمُ ٢ : ١٢٨ : ١٧
إِنَّ اللَّهَ بِالنَّاسِ لَرَءُوفٌ رَّحِيمٌ ٢ : ١٤٣ : ٤٥
وَأَنَا التَّوَّابُ الرَّحِيمُ ٢ : ١٦٠ : ١١
لَّا إِلَٰهَ إِلَّا هُوَ الرَّحْمَٰنُ الرَّحِيمُ ٢ : ١٦٣ : ٩
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٢ : ١٧٣ : ٢٥
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٢ : ١٨٢ : ١٦
فَإِنِ انتَهَوْا فَإِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٢ : ١٩٢ : ٦
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٢ : ١٩٩ : ١٢
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٢ : ٢١٨ : ١٦
فَإِن فَاءُوا فَإِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٢ : ٢٢٦ : ١٣
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٣ : ٣١ : ١٤
إِلَّا الَّذِينَ تَابُوا مِن بَعْدِ ذَٰلِكَ وَأَصْلَحُوا فَإِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٣ : ٨٩ : ١١
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٣ : ١٢٩ : ١٦
إِنَّ اللَّهَ كَانَ تَوَّابًا رَّحِيمًا ٤ : ١٦ : ١٤
إِنَّ اللَّهَ كَانَ غَفُورًا رَّحِيمًا ٤ : ٢٣ : ٥٤
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٤ : ٢٥ : ٥٨
إِنَّ اللَّهَ كَانَ بِكُمْ رَحِيمًا ٤ : ٢٩ : ٢٣
وَلَوْ أَنَّهُمْ إِذ ظَّلَمُوا أَنفُسَهُمْ جَاءُوكَ فَاسْتَغْفَرُوا اللَّهَ وَاسْتَغْفَرَ لَهُمُ الرَّسُولُ لَوَجَدُوا اللَّهَ تَوَّابًا رَّحِيمًا ٤ : ٦٤ : ٢٣
وَكَانَ اللَّهُ غَفُورًا رَّحِيمًا ٤ : ٩٦ : ٨
وَكَانَ اللَّهُ غَفُورًا رَّحِيمًا ٤ : ١٠٠ : ٣١
إِنَّ اللَّهَ كَانَ غَفُورًا رَّحِيمًا ٤ : ١٠٦ : ٧
وَمَن يَعْمَلْ سُوءًا أَوْ يَظْلِمْ نَفْسَهُ ثُمَّ يَسْتَغْفِرِ اللَّهَ يَجِدِ اللَّهَ غَفُورًا رَّحِيمًا ٤ : ١١٠ : ١٣
وَإِن تُصْلِحُوا وَتَتَّقُوا فَإِنَّ اللَّهَ كَانَ غَفُورًا رَّحِيمًا ٤ : ١٢٩ : ٢٢
وَكَانَ اللَّهُ غَفُورًا رَّحِيمًا ٤ : ١٥٢ : ١٧
فَمَنِ اضْطُرَّ فِي مَخْمَصَةٍ غَيْرَ مُتَجَانِفٍ لِّإِثْمٍ ۙ فَإِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٥ : ٣ : ٦١
فَاعْلَمُوا أَنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٥ : ٣٤ : ١٣
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٥ : ٣٩ : ١٤
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٥ : ٧٤ : ٨
اعْلَمُوا أَنَّ اللَّهَ شَدِيدُ الْعِقَابِ وَأَنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٥ : ٩٨ : ٩
أَنَّهُ مَنْ عَمِلَ مِنكُمْ سُوءًا بِجَهَالَةٍ ثُمَّ تَابَ مِن بَعْدِهِ وَأَصْلَحَ فَأَنَّهُ غَفُورٌ رَّحِيمٌ ٦ : ٥٤ : ٢٧
فَمَنِ اضْطُرَّ غَيْرَ بَاغٍ وَلَا عَادٍ فَإِنَّ رَبَّكَ غَفُورٌ رَّحِيمٌ ٦ : ١٤٥ : ٣٩
إِنَّ رَبَّكَ سَرِيعُ الْعِقَابِ وَإِنَّهُ لَغَفُورٌ رَّحِيمٌ ٦ : ١٦٥ : ٢١
وَالَّذِينَ عَمِلُوا السَّيِّئَاتِ ثُمَّ تَابُوا مِن بَعْدِهَا وَآمَنُوا إِنَّ رَبَّكَ مِن بَعْدِهَا لَغَفُورٌ رَّحِيمٌ ٧ : ١٥٣ : ١٤
وَإِنَّهُ لَغَفُورٌ رَّحِيمٌ ٧ : ١٦٧ : ١٩
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٨ : ٦٩ : ١١
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٨ : ٧٠ : ٢٤
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٩ : ٥ : ٢٦
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٩ : ٢٧ : ١٢
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٩ : ٩١ : ٢٦
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٩ : ٩٩ : ٢٧
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٩ : ١٠٢ : ١٧
أَلَمْ يَعْلَمُوا أَنَّ اللَّهَ هُوَ يَقْبَلُ التَّوْبَةَ عَنْ عِبَادِهِ وَيَأْخُذُ الصَّدَقَاتِ وَأَنَّ اللَّهَ هُوَ التَّوَّابُ الرَّحِيمُ ٩ : ١٠٤ : ١٦
إِنَّهُ بِهِمْ رَءُوفٌ رَّحِيمٌ ٩ : ١١٧ : ٢٧
إِنَّ اللَّهَ هُوَ التَّوَّابُ الرَّحِيمُ ٩ : ١١٨ : ٣١
لَقَدْ جَاءَكُمْ رَسُولٌ مِّنْ أَنفُسِكُمْ عَزِيزٌ عَلَيْهِ مَا عَنِتُّمْ حَرِيصٌ عَلَيْكُم بِالْمُؤْمِنِينَ رَءُوفٌ رَّحِيمٌ ٩ : ١٢٨ : ١٤
وَهُوَ الْغَفُورُ الرَّحِيمُ ١٠ : ١٠٧ : ٢٤
إِنَّ رَبِّي لَغَفُورٌ رَّحِيمٌ ١١ : ٤١ : ١١
إِنَّ رَبِّي رَحِيمٌ وَدُودٌ ١١ : ٩٠ : ٨
إِنَّ رَبِّي غَفُورٌ رَّحِيمٌ ١٢ : ٥٣ : ١٥
إِنَّهُ هُوَ الْغَفُورُ الرَّحِيمُ ١٢ : ٩٨ : ٩
وَمَنْ عَصَانِي فَإِنَّكَ غَفُورٌ رَّحِيمٌ ١٤ : ٣٦ : ١٥
نَبِّئْ عِبَادِي أَنِّي أَنَا الْغَفُورُ الرَّحِيمُ ١٥ : ٤٩ : ٦
إِنَّ رَبَّكُمْ لَرَءُوفٌ رَّحِيمٌ ١٦ : ٧ : ١٤
إِنَّ اللَّهَ لَغَفُورٌ رَّحِيمٌ ١٦ : ١٨ : ١٠
أَوْ يَأْخُذَهُمْ عَلَىٰ تَخَوُّفٍ فَإِنَّ رَبَّكُمْ لَرَءُوفٌ رَّحِيمٌ ١٦ : ٤٧ : ٨
ثُمَّ إِنَّ رَبَّكَ لِلَّذِينَ هَاجَرُوا مِن بَعْدِ مَا فُتِنُوا ثُمَّ جَاهَدُوا وَصَبَرُوا إِنَّ رَبَّكَ مِن بَعْدِهَا لَغَفُورٌ رَّحِيمٌ ١٦ : ١١٠ : ١٨
فَمَنِ اضْطُرَّ غَيْرَ بَاغٍ وَلَا عَادٍ فَإِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ١٦ : ١١٥ : ٢٢
ثُمَّ إِنَّ رَبَّكَ لِلَّذِينَ عَمِلُوا السُّوءَ بِجَهَالَةٍ ثُمَّ تَابُوا مِن بَعْدِ ذَٰلِكَ وَأَصْلَحُوا إِنَّ رَبَّكَ مِن بَعْدِهَا لَغَفُورٌ رَّحِيمٌ ١٦ : ١١٩ : ١٩
إِنَّهُ كَانَ بِكُمْ رَحِيمًا ١٧ : ٦٦ : ١٤
إِنَّ اللَّهَ بِالنَّاسِ لَرَءُوفٌ رَّحِيمٌ ٢٢ : ٦٥ : ٢٧
إِلَّا الَّذِينَ تَابُوا مِن بَعْدِ ذَٰلِكَ وَأَصْلَحُوا فَإِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٢٤ : ٥ : ١١

إِنَّهُ كَانَ غَفُورًا رَّحِيمًا ٢٥ : ٦ : ١٢

وَإِنَّ رَبَّكَ لَهُوَ الْعَزِيزُ الرَّحِيمُ ٢٦ : ٦٨ : ٥
وَإِنَّ رَبَّكَ لَهُوَ الْعَزِيزُ الرَّحِيمُ ٢٦ : ١٠٤ : ٥
وَإِنَّ رَبَّكَ لَهُوَ الْعَزِيزُ الرَّحِيمُ ٢٦ : ١٢٢ : ٥
وَإِنَّ رَبَّكَ لَهُوَ الْعَزِيزُ الرَّحِيمُ ٢٦ : ١٤٠ : ٥
وَإِنَّ رَبَّكَ لَهُوَ الْعَزِيزُ الرَّحِيمُ ٢٦ : ١٥٩ : ٥
وَإِنَّ رَبَّكَ لَهُوَ الْعَزِيزُ الرَّحِيمُ ٢٦ : ١٧٥ : ٥
وَإِنَّ رَبَّكَ لَهُوَ الْعَزِيزُ الرَّحِيمُ ٢٦ : ١٩١ : ٥
وَتَوَكَّلْ عَلَى الْعَزِيزِ الرَّحِيمِ ٢٦ : ٢١٧ : ٤
إِلَّا مَن ظَلَمَ ثُمَّ بَدَّلَ حُسْنًا بَعْدَ سُوءٍ فَإِنِّي غَفُورٌ رَّحِيمٌ ٢٧ : ١١ : ١١
إِنَّهُ مِن سُلَيْمَانَ وَإِنَّهُ بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ ٢٧ : ٣٠ : ٨
إِنَّهُ هُوَ الْغَفُورُ الرَّحِيمُ ٢٨ : ١٦ : ١٣
وَهُوَ الْعَزِيزُ الرَّحِيمُ ٣٠ : ٥ : ٨
ذَٰلِكَ عَالِمُ الْغَيْبِ وَالشَّهَادَةِ الْعَزِيزُ الرَّحِيمُ ٣٢ : ٦ : ٦
وَكَانَ اللَّهُ غَفُورًا رَّحِيمًا ٣٣ : ٥ : ٢٨
إِنَّ اللَّهَ كَانَ غَفُورًا رَّحِيمًا ٣٣ : ٢٤ : ١٦
وَكَانَ بِالْمُؤْمِنِينَ رَحِيمًا ٣٣ : ٤٣ : ١٣
وَكَانَ اللَّهُ غَفُورًا رَّحِيمًا ٣٣ : ٥٠ : ٦١
وَكَانَ اللَّهُ غَفُورًا رَّحِيمًا ٣٣ : ٥٩ : ٢١
وَكَانَ اللَّهُ غَفُورًا رَّحِيمًا ٣٣ : ٧٣ : ١٥
وَهُوَ الرَّحِيمُ الْغَفُورُ ٣٤ : ٢ : ١٧
تَنزِيلَ الْعَزِيزِ الرَّحِيمِ ٣٦ : ٥ : ٣
سَلَامٌ قَوْلًا مِّن رَّبٍّ رَّحِيمٍ ٣٦ : ٥٨ : ٥
إِنَّهُ هُوَ الْغَفُورُ الرَّحِيمُ ٣٩ : ٥٣ : ٢٠
تَنزِيلٌ مِّنَ الرَّحْمَٰنِ الرَّحِيمِ ٤١ : ٢ : ٤
نُزُلًا مِّنْ غَفُورٍ رَّحِيمٍ ٤١ : ٣٢ : ٤
أَلَا إِنَّ اللَّهَ هُوَ الْغَفُورُ الرَّحِيمُ ٤٢ : ٥ : ١٩
إِنَّهُ هُوَ الْعَزِيزُ الرَّحِيمُ ٤٤ : ٤٢ : ٨
وَهُوَ الْغَفُورُ الرَّحِيمُ ٤٦ : ٨ : ٢٥
وَكَانَ اللَّهُ غَفُورًا رَّحِيمًا ٤٨ : ١٤ : ١٤
وَالَّذِينَ مَعَهُ أَشِدَّاءُ عَلَى الْكُفَّارِ رُحَمَاءُ بَيْنَهُمْ ٤٨ : ٢٩ : ٩
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٤٩ : ٥ : ١٢
إِنَّ اللَّهَ تَوَّابٌ رَّحِيمٌ ٤٩ : ١٢ : ٣١
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٤٩ : ١٤ : ٢٧
إِنَّهُ هُوَ الْبَرُّ الرَّحِيمُ ٥٢ : ٢٨ : ٩
وَإِنَّ اللَّهَ بِكُمْ لَرَءُوفٌ رَّحِيمٌ ٥٧ : ٩ : ١٧
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٥٧ : ٢٨ : ٢١
فَإِن لَّمْ تَجِدُوا فَإِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٥٨ : ١٢ : ٢٢
وَالَّذِينَ جَاءُوا مِن بَعْدِهِمْ يَقُولُونَ رَبَّنَا اغْفِرْ لَنَا وَلِإِخْوَانِنَا الَّذِينَ سَبَقُونَا بِالْإِيمَانِ وَلَا تَجْعَلْ فِي قُلُوبِنَا غِلًّا لِّلَّذِينَ آمَنُوا رَبَّنَا إِنَّكَ رَءُوفٌ رَّحِيمٌ ٥٩ : ١٠ : ٢٣
هُوَ الرَّحْمَٰنُ الرَّحِيمُ ٥٩ : ٢٢ : ١٣
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٦٠ : ٧ : ١٥
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٦٠ : ١٢ : ٣٨
وَإِن تَعْفُوا وَتَصْفَحُوا وَتَغْفِرُوا فَإِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٦٤ : ١٤ : ١٨
وَاللَّهُ غَفُورٌ رَّحِيمٌ ٦٦ : ١ : ١٤
إِنَّ اللَّهَ غَفُورٌ رَّحِيمٌ ٧٣ : ٢٠ : ٧٨

اسم بصيغة «رَحْمَة»

فَلَوْلَا فَضْلُ اللَّهِ عَلَيْكُمْ وَرَحْمَتُهُ لَكُنتُم مِّنَ الْخَاسِرِينَ ٢ : ٦٤ : ١٠
وَاللَّهُ يَخْتَصُّ بِرَحْمَتِهِ مَن يَشَاءُ ٢ : ١٠٥ : ١٩
أُولَٰئِكَ عَلَيْهِمْ صَلَوَاتٌ مِّن رَّبِّهِمْ وَرَحْمَةٌ ٢ : ١٥٧ : ٦
ذَٰلِكَ تَخْفِيفٌ مِّن رَّبِّكُمْ وَرَحْمَةٌ ٢ : ١٧٨ : ٣٠
إِنَّ الَّذِينَ آمَنُوا وَالَّذِينَ هَاجَرُوا وَجَاهَدُوا فِي سَبِيلِ اللَّهِ أُولَٰئِكَ يَرْجُونَ رَحْمَتَ اللَّهِ ٢ : ٢١٨ : ١٢
رَبَّنَا لَا تُزِغْ قُلُوبَنَا بَعْدَ إِذْ هَدَيْتَنَا وَهَبْ لَنَا مِن لَّدُنكَ رَحْمَةً ٣ : ٨ : ١٢
يَخْتَصُّ بِرَحْمَتِهِ مَن يَشَاءُ ٣ : ٧٤ : ٢
وَأَمَّا الَّذِينَ ابْيَضَّتْ وُجُوهُهُمْ فَفِي رَحْمَةِ اللَّهِ هُمْ فِيهَا خَالِدُونَ ٣ : ١٠٧ : ٦
وَلَئِن قُتِلْتُمْ فِي سَبِيلِ اللَّهِ أَوْ مُتُّمْ لَمَغْفِرَةٌ مِّنَ اللَّهِ وَرَحْمَةٌ خَيْرٌ مِّمَّا يَجْمَعُونَ ٣ : ١٥٧ : ١١
فَبِمَا رَحْمَةٍ مِّنَ اللَّهِ لِنتَ لَهُمْ ٣ : ١٥٩ : ٢
وَلَوْلَا فَضْلُ اللَّهِ عَلَيْكُمْ وَرَحْمَتُهُ لَاتَّبَعْتُمُ الشَّيْطَانَ إِلَّا قَلِيلًا ٤ : ٨٣ : ٢٦
دَرَجَاتٍ مِّنْهُ وَمَغْفِرَةً وَرَحْمَةً ٤ : ٩٦ : ٤
وَلَوْلَا فَضْلُ اللَّهِ عَلَيْكَ وَرَحْمَتُهُ لَهَمَّت طَّائِفَةٌ مِّنْهُمْ أَن يُضِلُّوكَ وَمَا يُضِلُّونَ إِلَّا أَنفُسَهُمْ ٤ : ١١٣ : ٥
فَأَمَّا الَّذِينَ آمَنُوا بِاللَّهِ وَاعْتَصَمُوا بِهِ فَسَيُدْخِلُهُمْ فِي رَحْمَةٍ مِّنْهُ وَفَضْلٍ وَيَهْدِيهِمْ إِلَيْهِ صِرَاطًا مُّسْتَقِيمًا ٤ : ١٧٥ : ٩
كَتَبَ عَلَىٰ نَفْسِهِ الرَّحْمَةَ ٦ : ١٢ : ١٢
كَتَبَ رَبُّكُمْ عَلَىٰ نَفْسِهِ الرَّحْمَةَ ٦ : ٥٤ : ١٣
وَرَبُّكَ الْغَنِيُّ ذُو الرَّحْمَةِ ٦ : ١٣٣ : ٤
فَإِن كَذَّبُوكَ فَقُل رَّبُّكُمْ ذُو رَحْمَةٍ وَاسِعَةٍ وَلَا يُرَدُّ بَأْسُهُ عَنِ الْقَوْمِ الْمُجْرِمِينَ ٦ : ١٤٧ : ٦
ثُمَّ آتَيْنَا مُوسَى الْكِتَابَ تَمَامًا عَلَى الَّذِي أَحْسَنَ وَتَفْصِيلًا لِّكُلِّ شَيْءٍ وَهُدًى وَرَحْمَةً لَّعَلَّهُم بِلِقَاءِ رَبِّهِمْ يُؤْمِنُونَ ٦ : ١٥٤ : ١٣
فَقَدْ جَاءَكُم بَيِّنَةٌ مِّن رَّبِّكُمْ وَهُدًى وَرَحْمَةٌ ٦ : ١٥٧ : ١٧
أَهَٰؤُلَاءِ الَّذِينَ أَقْسَمْتُمْ لَا يَنَالُهُمُ اللَّهُ بِرَحْمَةٍ ٧ : ٤٩ : ٧
وَلَقَدْ جِئْنَاهُم بِكِتَابٍ فَصَّلْنَاهُ عَلَىٰ عِلْمٍ هُدًى وَرَحْمَةً لِّقَوْمٍ يُؤْمِنُونَ ٧ : ٥٢ : ٨
إِنَّ رَحْمَتَ اللَّهِ قَرِيبٌ مِّنَ الْمُحْسِنِينَ ٧ : ٥٦ : ١١
وَهُوَ الَّذِي يُرْسِلُ الرِّيَاحَ بُشْرًا بَيْنَ يَدَيْ رَحْمَتِهِ ٧ : ٥٧ : ٨
فَأَنجَيْنَاهُ وَالَّذِينَ مَعَهُ بِرَحْمَةٍ مِّنَّا وَقَطَعْنَا دَابِرَ الَّذِينَ كَذَّبُوا بِآيَاتِنَا ٧ : ٧٢ : ٤
قَالَ رَبِّ اغْفِرْ لِي وَلِأَخِي وَأَدْخِلْنَا فِي رَحْمَتِكَ ٧ : ١٥١ : ٨
وَفِي نُسْخَتِهَا هُدًى وَرَحْمَةٌ لِّلَّذِينَ هُمْ لِرَبِّهِمْ يَرْهَبُونَ ٧ : ١٥٤ : ١١
وَرَحْمَتِي وَسِعَتْ كُلَّ شَيْءٍ ٧ : ١٥٦ : ١٨
هَٰذَا بَصَائِرُ مِن رَّبِّكُمْ وَهُدًى وَرَحْمَةٌ لِّقَوْمٍ يُؤْمِنُونَ ٧ : ٢٠٣ : ٢١
يُبَشِّرُهُمْ رَبُّهُم بِرَحْمَةٍ مِّنْهُ وَرِضْوَانٍ وَجَنَّاتٍ لَّهُمْ فِيهَا نَعِيمٌ مُّقِيمٌ ٩ : ٢١ : ٣
قُلْ أُذُنُ خَيْرٍ لَّكُمْ يُؤْمِنُ بِاللَّهِ وَيُؤْمِنُ لِلْمُؤْمِنِينَ وَرَحْمَةٌ لِّلَّذِينَ آمَنُوا مِنكُمْ ٩ : ٦١ : ١٦
سَيُدْخِلُهُمُ اللَّهُ فِي رَحْمَتِهِ ٩ : ٩٩ : ٢٣
وَإِذَا أَذَقْنَا النَّاسَ رَحْمَةً مِّن بَعْدِ ضَرَّاءَ مَسَّتْهُمْ إِذَا لَهُم مَّكْرٌ فِي آيَاتِنَا ١٠ : ٢١ : ٤
يَا أَيُّهَا النَّاسُ قَدْ جَاءَتْكُم مَّوْعِظَةٌ مِّن رَّبِّكُمْ وَشِفَاءٌ لِّمَا فِي الصُّدُورِ وَهُدًى وَرَحْمَةٌ لِّلْمُؤْمِنِينَ ١٠ : ٥٧ : ١٣
قُلْ بِفَضْلِ اللَّهِ وَبِرَحْمَتِهِ فَبِذَٰلِكَ فَلْيَفْرَحُوا هُوَ خَيْرٌ مِّمَّا يَجْمَعُونَ ١٠ : ٥٨ : ٤
وَنَجِّنَا بِرَحْمَتِكَ مِنَ الْقَوْمِ الْكَافِرِينَ ١٠ : ٨٦ : ٢
وَلَئِنْ أَذَقْنَا الْإِنسَانَ مِنَّا رَحْمَةً ثُمَّ نَزَعْنَاهَا مِنْهُ إِنَّهُ لَيَئُوسٌ كَفُورٌ ١١ : ٩ : ٥
أَفَمَن كَانَ عَلَىٰ بَيِّنَةٍ مِّن رَّبِّهِ وَيَتْلُوهُ شَاهِدٌ مِّنْهُ وَمِن قَبْلِهِ كِتَابُ مُوسَىٰ إِمَامًا وَرَحْمَةً ١١ : ١٧ : ١٥
قَالَ يَا قَوْمِ أَرَأَيْتُمْ إِن كُنتُ عَلَىٰ بَيِّنَةٍ مِّن رَّبِّي وَآتَانِي رَحْمَةً مِّنْ عِندِهِ فَعُمِّيَتْ عَلَيْكُمْ أَنُلْزِمُكُمُوهَا وَأَنتُمْ لَهَا كَارِهُونَ ١١ : ٢٨ : ١١
وَلَمَّا جَاءَ أَمْرُنَا نَجَّيْنَا هُودًا وَالَّذِينَ آمَنُوا مَعَهُ بِرَحْمَةٍ مِّنَّا وَنَجَّيْنَاهُم مِّنْ عَذَابٍ غَلِيظٍ ١١ : ٥٨ : ٩
قَالَ يَا قَوْمِ أَرَأَيْتُمْ إِن كُنتُ عَلَىٰ بَيِّنَةٍ مِّن رَّبِّي وَآتَانِي مِنْهُ رَحْمَةً فَمَن يَنصُرُنِي مِنَ اللَّهِ إِنْ عَصَيْتُهُ ١١ : ٦٣ : ١٢
فَلَمَّا جَاءَ أَمْرُنَا نَجَّيْنَا صَالِحًا وَالَّذِينَ آمَنُوا مَعَهُ بِرَحْمَةٍ مِّنَّا وَمِنْ خِزْيِ يَوْمِئِذٍ ١١ : ٦٦ : ٩
رَحْمَتُ اللَّهِ وَبَرَكَاتُهُ عَلَيْكُمْ أَهْلَ الْبَيْتِ ١١ : ٧٣ : ٦
وَلَمَّا جَاءَ أَمْرُنَا نَجَّيْنَا شُعَيْبًا وَالَّذِينَ آمَنُوا مَعَهُ بِرَحْمَةٍ مِّنَّا وَأَخَذَتِ الَّذِينَ ظَلَمُوا الصَّيْحَةُ فَأَصْبَحُوا فِي دِيَارِهِمْ جَاثِمِينَ ١١ : ٩٤ : ٩
نُصِيبُ بِرَحْمَتِنَا مَن نَّشَاءُ ١٢ : ٥٦ : ١١
مَا كَانَ حَدِيثًا يُفْتَرَىٰ وَلَٰكِن تَصْدِيقَ الَّذِي بَيْنَ يَدَيْهِ وَتَفْصِيلَ كُلِّ شَيْءٍ وَهُدًى وَرَحْمَةً لِّقَوْمٍ يُؤْمِنُونَ ١٢ : ١١١ : ٢١
قَالَ وَمَن يَقْنَطُ مِن رَّحْمَةِ رَبِّهِ إِلَّا الضَّالُّونَ ١٥ : ٥٦ : ٥
وَمَا أَنزَلْنَا عَلَيْكَ الْكِتَابَ إِلَّا لِتُبَيِّنَ لَهُمُ الَّذِي اخْتَلَفُوا فِيهِ ۙ وَهُدًى وَرَحْمَةً لِّقَوْمٍ يُؤْمِنُونَ ١٦ : ٦٤ : ١٢
وَنَزَّلْنَا عَلَيْكَ الْكِتَابَ تِبْيَانًا لِّكُلِّ شَيْءٍ وَهُدًى وَرَحْمَةً وَبُشْرَىٰ لِلْمُسْلِمِينَ ١٦ : ٨٩ : ٢٢
وَاخْفِضْ لَهُمَا جَنَاحَ الذُّلِّ مِنَ الرَّحْمَةِ وَقُل رَّبِّ ارْحَمْهُمَا كَمَا رَبَّيَانِي صَغِيرًا ١٧ : ٢٤ : ٦
وَإِمَّا تُعْرِضَنَّ عَنْهُمُ ابْتِغَاءَ رَحْمَةٍ مِّن رَّبِّكَ تَرْجُوهَا فَقُل لَّهُمْ قَوْلًا مَّيْسُورًا ١٧ : ٢٨ : ٥
أُولَٰئِكَ الَّذِينَ يَدْعُونَ يَبْتَغُونَ إِلَىٰ رَبِّهِمُ الْوَسِيلَةَ أَيُّهُمْ أَقْرَبُ وَيَرْجُونَ رَحْمَتَهُ وَيَخَافُونَ عَذَابَهُ ١٧ : ٥٧ : ١١
وَنُنَزِّلُ مِنَ الْقُرْآنِ مَا هُوَ شِفَاءٌ وَرَحْمَةٌ لِّلْمُؤْمِنِينَ ۙ وَلَا يَزِيدُ الظَّالِمِينَ إِلَّا خَسَارًا ١٧ : ٨٢ : ٧
إِلَّا رَحْمَةً مِّن رَّبِّكَ ١٧ : ٨٧ : ٢
قُل لَّوْ أَنتُمْ تَمْلِكُونَ خَزَائِنَ رَحْمَةِ رَبِّي إِذًا لَّأَمْسَكْتُمْ خَشْيَةَ الْإِنفَاقِ ١٧ : ١٠٠ : ٦
إِذْ أَوَى الْفِتْيَةُ إِلَى الْكَهْفِ فَقَالُوا رَبَّنَا آتِنَا مِن لَّدُنكَ رَحْمَةً وَهَيِّئْ لَنَا مِنْ أَمْرِنَا رَشَدًا ١٨ : ١٠ : ١١
وَإِذِ اعْتَزَلْتُمُوهُمْ وَمَا يَعْبُدُونَ إِلَّا اللَّهَ فَأْوُوا إِلَى الْكَهْفِ يَنشُرْ لَكُمْ رَبُّكُم مِّن رَّحْمَتِهِ وَيُهَيِّئْ لَكُم مِّنْ أَمْرِكُم مِّرْفَقًا ١٨ : ١٦ : ١٤
وَرَبُّكَ الْغَفُورُ ذُو الرَّحْمَةِ ١٨ : ٥٨ : ٤
فَوَجَدَا عَبْدًا مِّنْ عِبَادِنَا آتَيْنَاهُ رَحْمَةً مِّنْ عِندِنَا وَعَلَّمْنَاهُ مِن لَّدُنَّا عِلْمًا ١٨ : ٦٥ : ٦
وَأَمَّا الْجِدَارُ فَكَانَ لِغُلَامَيْنِ يَتِيمَيْنِ فِي الْمَدِينَةِ وَكَانَ تَحْتَهُ كَنزٌ لَّهُمَا وَكَانَ أَبُوهُمَا صَالِحًا فَأَرَادَ رَبُّكَ أَن يَبْلُغَا أَشُدَّهُمَا وَيَسْتَخْرِجَا كَنزَهُمَا رَحْمَةً مِّن رَّبِّكَ ١٨ : ٨٢ : ٢٢
قَالَ هَٰذَا رَحْمَةٌ مِّن رَّبِّي ١٨ : ٩٨ : ٣
ذِكْرُ رَحْمَتِ رَبِّكَ عَبْدَهُ زَكَرِيَّا ١٩ : ٢ : ٢
وَلِنَجْعَلَهُ آيَةً لِّلنَّاسِ وَرَحْمَةً مِّنَّا ١٩ : ٢١ : ١١
وَوَهَبْنَا لَهُم مِّن رَّحْمَتِنَا وَجَعَلْنَا لَهُمْ لِسَانَ صِدْقٍ عَلِيًّا ١٩ : ٥٠ : ٤
وَوَهَبْنَا لَهُ مِن رَّحْمَتِنَا أَخَاهُ هَارُونَ نَبِيًّا ١٩ : ٥٣ : ٤
وَأَدْخَلْنَاهُ فِي رَحْمَتِنَا ٢١ : ٧٥ : ٣
وَآتَيْنَاهُ أَهْلَهُ وَمِثْلَهُم مَّعَهُمْ رَحْمَةً مِّنْ عِندِنَا وَذِكْرَىٰ لِلْعَابِدِينَ ٢١ : ٨٤ : ١٢
وَأَدْخَلْنَاهُمْ فِي رَحْمَتِنَا ٢١ : ٨٦ : ٣
وَمَا أَرْسَلْنَاكَ إِلَّا رَحْمَةً لِّلْعَالَمِينَ ٢١ : ١٠٧ : ٤
وَلَوْلَا فَضْلُ اللَّهِ عَلَيْكُمْ وَرَحْمَتُهُ وَأَنَّ اللَّهَ تَوَّابٌ حَكِيمٌ ٢٤ : ١٠ : ٥
وَلَوْلَا فَضْلُ اللَّهِ عَلَيْكُمْ وَرَحْمَتُهُ فِي الدُّنْيَا وَالْآخِرَةِ لَمَسَّكُمْ فِي مَا أَفَضْتُمْ فِيهِ عَذَابٌ عَظِيمٌ ٢٤ : ١٤ : ٥
وَلَوْلَا فَضْلُ اللَّهِ عَلَيْكُمْ وَرَحْمَتُهُ وَأَنَّ اللَّهَ رَءُوفٌ رَّحِيمٌ ٢٤ : ٢٠ : ٥
وَلَوْلَا فَضْلُ اللَّهِ عَلَيْكُمْ وَرَحْمَتُهُ مَا زَكَىٰ مِنكُم مِّنْ أَحَدٍ أَبَدًا وَلَٰكِنَّ اللَّهَ يُزَكِّي مَن يَشَاءُ ٢٤ : ٢١ : ٢٠
وَهُوَ الَّذِي أَرْسَلَ الرِّيَاحَ بُشْرًا بَيْنَ يَدَيْ رَحْمَتِهِ ٢٥ : ٤٨ : ٨
فَتَبَسَّمَ ضَاحِكًا مِّن قَوْلِهَا وَقَالَ رَبِّ أَوْزِعْنِي أَنْ أَشْكُرَ نِعْمَتَكَ الَّتِي أَنْعَمْتَ عَلَيَّ وَعَلَىٰ وَالِدَيَّ وَأَنْ أَعْمَلَ صَالِحًا تَرْضَاهُ وَأَدْخِلْنِي بِرَحْمَتِكَ فِي عِبَادِكَ الصَّالِحِينَ ٢٧ : ١٩ : ٢١
أَمَّن يَهْدِيكُمْ فِي ظُلُمَاتِ الْبَرِّ وَالْبَحْرِ وَمَن يُرْسِلُ الرِّيَاحَ بُشْرًا بَيْنَ يَدَيْ رَحْمَتِهِ ٢٧ : ٦٣ : ١٣
وَإِنَّهُ لَهُدًى وَرَحْمَةٌ لِّلْمُؤْمِنِينَ ٢٧ : ٧٧ : ٣
وَلَقَدْ آتَيْنَا مُوسَى الْكِتَابَ مِن بَعْدِ مَا أَهْلَكْنَا الْقُرُونَ الْأُولَىٰ بَصَائِرَ لِلنَّاسِ وَهُدًى وَرَحْمَةً لَّعَلَّهُمْ يَتَذَكَّرُونَ ٢٨ : ٤٣ : ١٤
وَمَا كُنتَ بِجَانِبِ الطُّورِ إِذْ نَادَيْنَا وَلَٰكِن رَّحْمَةً مِّن رَّبِّكَ لِتُنذِرَ قَوْمًا مَّا أَتَاهُم مِّن نَّذِيرٍ مِّن قَبْلِكَ لَعَلَّهُمْ يَتَذَكَّرُونَ ٢٨ : ٤٦ : ٨
وَمِن رَّحْمَتِهِ جَعَلَ لَكُمُ اللَّيْلَ وَالنَّهَارَ لِتَسْكُنُوا فِيهِ وَلِتَبْتَغُوا مِن فَضْلِهِ وَلَعَلَّكُمْ تَشْكُرُونَ ٢٨ : ٧٣ : ٢
وَمَا كُنتَ تَرْجُو أَن يُلْقَىٰ إِلَيْكَ الْكِتَابُ إِلَّا رَحْمَةً مِّن رَّبِّكَ ٢٨ : ٨٦ : ٩
وَالَّذِينَ كَفَرُوا بِآيَاتِ اللَّهِ وَلِقَائِهِ أُولَٰئِكَ يَئِسُوا مِن رَّحْمَتِي وَأُولَٰئِكَ لَهُمْ عَذَابٌ أَلِيمٌ ٢٩ : ٢٣ : ٩
إِنَّ فِي ذَٰلِكَ لَرَحْمَةً وَذِكْرَىٰ لِقَوْمٍ يُؤْمِنُونَ ٢٩ : ٥١ : ١٢
وَمِنْ آيَاتِهِ أَنْ خَلَقَ لَكُم مِّنْ أَنفُسِكُمْ أَزْوَاجًا لِّتَسْكُنُوا إِلَيْهَا وَجَعَلَ بَيْنَكُم مَّوَدَّةً وَرَحْمَةً ٣٠ : ٢١ : ١٤
وَإِذَا مَسَّ النَّاسَ ضُرٌّ دَعَوْا رَبَّهُم مُّنِيبِينَ إِلَيْهِ ثُمَّ إِذَا أَذَاقَهُم مِّنْهُ رَحْمَةً إِذَا فَرِيقٌ مِّنْهُم بِرَبِّهِمْ يُشْرِكُونَ ٣٠ : ٣٣ : ١٣
وَإِذَا أَذَقْنَا النَّاسَ رَحْمَةً فَرِحُوا بِهَا ٣٠ : ٣٦ : ٤
وَمِنْ آيَاتِهِ أَن يُرْسِلَ الرِّيَاحَ مُبَشِّرَاتٍ وَلِيُذِيقَكُم مِّن رَّحْمَتِهِ وَلِتَجْرِيَ الْفُلْكُ بِأَمْرِهِ وَلِتَبْتَغُوا مِن فَضْلِهِ وَلَعَلَّكُمْ تَشْكُرُونَ ٣٠ : ٤٦ : ٩
فَانظُرْ إِلَىٰ آثَارِ رَحْمَتِ اللَّهِ كَيْفَ يُحْيِي الْأَرْضَ بَعْدَ مَوْتِهَا ٣٠ : ٥٠ : ٤
هُدًى وَرَحْمَةً لِّلْمُحْسِنِينَ ٣١ : ٣ : ٢
قُلْ مَن ذَا الَّذِي يَعْصِمُكُم مِّنَ اللَّهِ إِنْ أَرَادَ بِكُمْ سُوءًا أَوْ أَرَادَ بِكُمْ رَحْمَةً ٣٣ : ١٧ : ١٥
مَّا يَفْتَحِ اللَّهُ لِلنَّاسِ مِن رَّحْمَةٍ فَلَا مُمْسِكَ لَهَا ٣٥ : ٢ : ٦
إِلَّا رَحْمَةً مِّنَّا وَمَتَاعًا إِلَىٰ حِينٍ ٣٦ : ٤٤ : ٢
أَمْ عِندَهُمْ خَزَائِنُ رَحْمَةِ رَبِّكَ الْعَزِيزِ الْوَهَّابِ ٣٨ : ٩ : ٤
وَوَهَبْنَا لَهُ أَهْلَهُ وَمِثْلَهُم مَّعَهُمْ رَحْمَةً مِّنَّا وَذِكْرَىٰ لِأُولِي الْأَلْبَابِ ٣٨ : ٤٣ : ٦
أَمَّنْ هُوَ قَانِتٌ آنَاءَ اللَّيْلِ سَاجِدًا وَقَائِمًا يَحْذَرُ الْآخِرَةَ وَيَرْجُو رَحْمَةَ رَبِّهِ ٣٩ : ٩ : ١١
قُلْ أَفَرَأَيْتُم مَّا تَدْعُونَ مِن دُونِ اللَّهِ إِنْ أَرَادَنِيَ اللَّهُ بِضُرٍّ هَلْ هُنَّ كَاشِفَاتُ ضُرِّهِ أَوْ أَرَادَنِي بِرَحْمَةٍ هَلْ هُنَّ مُمْسِكَاتُ رَحْمَتِهِ ٣٩ : ٣٨ : ٢٦
قُلْ أَفَرَأَيْتُم مَّا تَدْعُونَ مِن دُونِ اللَّهِ إِنْ أَرَادَنِيَ اللَّهُ بِضُرٍّ هَلْ هُنَّ كَاشِفَاتُ ضُرِّهِ أَوْ أَرَادَنِي بِرَحْمَةٍ هَلْ هُنَّ مُمْسِكَاتُ رَحْمَتِهِ ٣٩ : ٣٨ : ٣٠
قُلْ يَا عِبَادِيَ الَّذِينَ أَسْرَفُوا عَلَىٰ أَنفُسِهِمْ لَا تَقْنَطُوا مِن رَّحْمَةِ اللَّهِ ٣٩ : ٥٣ : ١٠
الَّذِينَ يَحْمِلُونَ الْعَرْشَ وَمَنْ حَوْلَهُ يُسَبِّحُونَ بِحَمْدِ رَبِّهِمْ وَيُؤْمِنُونَ بِهِ وَيَسْتَغْفِرُونَ لِلَّذِينَ آمَنُوا رَبَّنَا وَسِعْتَ كُلَّ شَيْءٍ رَّحْمَةً وَعِلْمًا فَاغْفِرْ لِلَّذِينَ تَابُوا وَاتَّبَعُوا سَبِيلَكَ وَقِهِمْ عَذَابَ الْجَحِيمِ ٤٠ : ٧ : ١٨
وَلَئِنْ أَذَقْنَاهُ رَحْمَةً مِّنَّا مِن بَعْدِ ضَرَّاءَ مَسَّتْهُ لَيَقُولَنَّ هَٰذَا لِي وَمَا أَظُنُّ السَّاعَةَ قَائِمَةً وَلَئِن رُّجِعْتُ إِلَىٰ رَبِّي إِنَّ لِي عِندَهُ لَلْحُسْنَىٰ ٤١ : ٥٠ : ٣
وَلَوْ شَاءَ اللَّهُ لَجَعَلَهُمْ أُمَّةً وَاحِدَةً وَلَٰكِن يُدْخِلُ مَن يَشَاءُ فِي رَحْمَتِهِ ٤٢ : ٨ : ١٢
وَهُوَ الَّذِي يُنَزِّلُ الْغَيْثَ مِن بَعْدِ مَا قَنَطُوا وَيَنشُرُ رَحْمَتَهُ ٤٢ : ٢٨ : ١٠
وَإِنَّا إِذَا أَذَقْنَا الْإِنسَانَ مِنَّا رَحْمَةً فَرِحَ بِهَا ٤٢ : ٤٨ : ١٦
أَهُمْ يَقْسِمُونَ رَحْمَتَ رَبِّكَ ٤٣ : ٣٢ : ٣
وَرَحْمَتُ رَبِّكَ خَيْرٌ مِّمَّا يَجْمَعُونَ ٤٣ : ٣٢ : ٢١
رَحْمَةً مِّن رَّبِّكَ ٤٤ : ٦ : ١
هَٰذَا بَصَائِرُ لِلنَّاسِ وَهُدًى وَرَحْمَةٌ لِّقَوْمٍ يُوقِنُونَ ٤٥ : ٢٠ : ٥
فَأَمَّا الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ فَيُدْخِلُهُمْ رَبُّهُمْ فِي رَحْمَتِهِ ٤٥ : ٣٠ : ٩
وَمِن قَبْلِهِ كِتَابُ مُوسَىٰ إِمَامًا وَرَحْمَةً ٤٦ : ١٢ : ٦
لِّيُدْخِلَ اللَّهُ فِي رَحْمَتِهِ مَن يَشَاءُ ٤٨ : ٢٥ : ٣٠
يَوْمَ يَقُولُ الْمُنَافِقُونَ وَالْمُنَافِقَاتُ لِلَّذِينَ آمَنُوا انظُرُونَا نَقْتَبِسْ مِن نُّورِكُمْ قِيلَ ارْجِعُوا وَرَاءَكُمْ فَالْتَمِسُوا نُورًا فَضُرِبَ بَيْنَهُم بِسُورٍ لَّهُ بَابٌ بَاطِنُهُ فِيهِ الرَّحْمَةُ وَظَاهِرُهُ مِن قِبَلِهِ الْعَذَابُ ٥٧ : ١٣ : ٢٣
ثُمَّ قَفَّيْنَا عَلَىٰ آثَارِهِم بِرُسُلِنَا وَقَفَّيْنَا بِعِيسَى ابْنِ مَرْيَمَ وَآتَيْنَاهُ الْإِنجِيلَ وَجَعَلْنَا فِي قُلُوبِ الَّذِينَ اتَّبَعُوهُ رَأْفَةً وَرَحْمَةً وَرَهْبَانِيَّةً ابْتَدَعُوهَا مَا كَتَبْنَاهَا عَلَيْهِمْ إِلَّا ابْتِغَاءَ رِضْوَانِ اللَّهِ فَمَا رَعَوْهَا حَقَّ رِعَايَتِهَا ٥٧ : ٢٧ : ١٨
يَا أَيُّهَا الَّذِينَ آمَنُوا اتَّقُوا اللَّهَ وَآمِنُوا بِرَسُولِهِ يُؤْتِكُمْ كِفْلَيْنِ مِن رَّحْمَتِهِ وَيَجْعَل لَّكُمْ نُورًا تَمْشُونَ بِهِ وَيَغْفِرْ لَكُمْ ٥٧ : ٢٨ : ١١
يُدْخِلُ مَن يَشَاءُ فِي رَحْمَتِهِ ٧٦ : ٣١ : ٥

اسم بصيغة «رَحْمٰن»

بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ ١ : ١ : ٣
الرَّحْمَٰنِ الرَّحِيمِ ١ : ٣ : ١
لَّا إِلَٰهَ إِلَّا هُوَ الرَّحْمَٰنُ الرَّحِيمُ ٢ : ١٦٣ : ٨
كَذَٰلِكَ أَرْسَلْنَاكَ فِي أُمَّةٍ قَدْ خَلَتْ مِن قَبْلِهَا أُمَمٌ لِّتَتْلُوَ عَلَيْهِمُ الَّذِي أَوْحَيْنَا إِلَيْكَ وَهُمْ يَكْفُرُونَ بِالرَّحْمَٰنِ ١٣ : ٣٠ : ١٧
قُلِ ادْعُوا اللَّهَ أَوِ ادْعُوا الرَّحْمَٰنَ ١٧ : ١١٠ : ٦
قَالَتْ إِنِّي أَعُوذُ بِالرَّحْمَٰنِ مِنكَ إِن كُنتَ تَقِيًّا ١٩ : ١٨ : ٤
فَإِمَّا تَرَيِنَّ مِنَ الْبَشَرِ أَحَدًا فَقُولِي إِنِّي نَذَرْتُ لِلرَّحْمَٰنِ صَوْمًا فَلَنْ أُكَلِّمَ الْيَوْمَ إِنسِيًّا ١٩ : ٢٦ : ١٣
إِنَّ الشَّيْطَانَ كَانَ لِلرَّحْمَٰنِ عَصِيًّا ١٩ : ٤٤ : ٨
يَا أَبَتِ إِنِّي أَخَافُ أَن يَمَسَّكَ عَذَابٌ مِّنَ الرَّحْمَٰنِ فَتَكُونَ لِلشَّيْطَانِ وَلِيًّا ١٩ : ٤٥ : ٨
إِذَا تُتْلَىٰ عَلَيْهِمْ آيَاتُ الرَّحْمَٰنِ خَرُّوا سُجَّدًا وَبُكِيًّا ۩ ١٩ : ٥٨ : ٢٦
جَنَّاتِ عَدْنٍ الَّتِي وَعَدَ الرَّحْمَٰنُ عِبَادَهُ بِالْغَيْبِ ١٩ : ٦١ : ٥
ثُمَّ لَنَنزِعَنَّ مِن كُلِّ شِيعَةٍ أَيُّهُمْ أَشَدُّ عَلَى الرَّحْمَٰنِ عِتِيًّا ١٩ : ٦٩ : ٩
قُلْ مَن كَانَ فِي الضَّلَالَةِ فَلْيَمْدُدْ لَهُ الرَّحْمَٰنُ مَدًّا ١٩ : ٧٥ : ٨
أَطَّلَعَ الْغَيْبَ أَمِ اتَّخَذَ عِندَ الرَّحْمَٰنِ عَهْدًا ١٩ : ٧٨ : ٦
يَوْمَ نَحْشُرُ الْمُتَّقِينَ إِلَى الرَّحْمَٰنِ وَفْدًا ١٩ : ٨٥ : ٥
لَّا يَمْلِكُونَ الشَّفَاعَةَ إِلَّا مَنِ اتَّخَذَ عِندَ الرَّحْمَٰنِ عَهْدًا ١٩ : ٨٧ : ٨
وَقَالُوا اتَّخَذَ الرَّحْمَٰنُ وَلَدًا ١٩ : ٨٨ : ٣
أَن دَعَوْا لِلرَّحْمَٰنِ وَلَدًا ١٩ : ٩١ : ٣
وَمَا يَنبَغِي لِلرَّحْمَٰنِ أَن يَتَّخِذَ وَلَدًا ١٩ : ٩٢ : ٣
إِن كُلُّ مَن فِي السَّمَاوَاتِ وَالْأَرْضِ إِلَّا آتِي الرَّحْمَٰنِ عَبْدًا ١٩ : ٩٣ : ٩
إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ سَيَجْعَلُ لَهُمُ الرَّحْمَٰنُ وُدًّا ١٩ : ٩٦ : ٨
الرَّحْمَٰنُ عَلَى الْعَرْشِ اسْتَوَىٰ ٢٠ : ٥ : ١
وَإِنَّ رَبَّكُمُ الرَّحْمَٰنُ فَاتَّبِعُونِي وَأَطِيعُوا أَمْرِي ٢٠ : ٩٠ : ١٣
وَخَشَعَتِ الْأَصْوَاتُ لِلرَّحْمَٰنِ فَلَا تَسْمَعُ إِلَّا هَمْسًا ٢٠ : ١٠٨ : ٩
يَوْمَئِذٍ لَّا تَنفَعُ الشَّفَاعَةُ إِلَّا مَنْ أَذِنَ لَهُ الرَّحْمَٰنُ وَرَضِيَ لَهُ قَوْلًا ٢٠ : ١٠٩ : ٩
وَقَالُوا اتَّخَذَ الرَّحْمَٰنُ وَلَدًا ٢١ : ٢٦ : ٣
وَإِذَا رَآكَ الَّذِينَ كَفَرُوا إِن يَتَّخِذُونَكَ إِلَّا هُزُوًا أَهَٰذَا الَّذِي يَذْكُرُ آلِهَتَكُمْ وَهُم بِذِكْرِ الرَّحْمَٰنِ هُمْ كَافِرُونَ ٢١ : ٣٦ : ١٥
قُلْ مَن يَكْلَؤُكُم بِاللَّيْلِ وَالنَّهَارِ مِنَ الرَّحْمَٰنِ ٢١ : ٤٢ : ٧
وَرَبُّنَا الرَّحْمَٰنُ الْمُسْتَعَانُ عَلَىٰ مَا تَصِفُونَ ٢١ : ١١٢ : ٦
الْمُلْكُ يَوْمَئِذٍ الْحَقُّ لِلرَّحْمَٰنِ ٢٥ : ٢٦ : ٤
الرَّحْمَٰنُ فَاسْأَلْ بِهِ خَبِيرًا ٢٥ : ٥٩ : ١٤
وَإِذَا قِيلَ لَهُمُ اسْجُدُوا لِلرَّحْمَٰنِ قَالُوا وَمَا الرَّحْمَٰنُ أَنَسْجُدُ لِمَا تَأْمُرُنَا وَزَادَهُمْ نُفُورًا ۩ ٢٥ : ٦٠ : ٥
وَإِذَا قِيلَ لَهُمُ اسْجُدُوا لِلرَّحْمَٰنِ قَالُوا وَمَا الرَّحْمَٰنُ أَنَسْجُدُ لِمَا تَأْمُرُنَا وَزَادَهُمْ نُفُورًا ۩ ٢٥ : ٦٠ : ٨
وَعِبَادُ الرَّحْمَٰنِ الَّذِينَ يَمْشُونَ عَلَى الْأَرْضِ هَوْنًا وَإِذَا خَاطَبَهُمُ الْجَاهِلُونَ قَالُوا سَلَامًا ٢٥ : ٦٣ : ٢
وَمَا يَأْتِيهِم مِّن ذِكْرٍ مِّنَ الرَّحْمَٰنِ مُحْدَثٍ إِلَّا كَانُوا عَنْهُ مُعْرِضِينَ ٢٦ : ٥ : ٦
إِنَّهُ مِن سُلَيْمَانَ وَإِنَّهُ بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ ٢٧ : ٣٠ : ٧
إِنَّمَا تُنذِرُ مَنِ اتَّبَعَ الذِّكْرَ وَخَشِيَ الرَّحْمَٰنَ بِالْغَيْبِ ٣٦ : ١١ : ٧
قَالُوا مَا أَنتُمْ إِلَّا بَشَرٌ مِّثْلُنَا وَمَا أَنزَلَ الرَّحْمَٰنُ مِن شَيْءٍ إِنْ أَنتُمْ إِلَّا تَكْذِبُونَ ٣٦ : ١٥ : ٩
أَأَتَّخِذُ مِن دُونِهِ آلِهَةً إِن يُرِدْنِ الرَّحْمَٰنُ بِضُرٍّ لَّا تُغْنِ عَنِّي شَفَاعَتُهُمْ شَيْئًا وَلَا يُنقِذُونِ ٣٦ : ٢٣ : ٧
هَٰذَا مَا وَعَدَ الرَّحْمَٰنُ وَصَدَقَ الْمُرْسَلُونَ ٣٦ : ٥٢ : ١٠
تَنزِيلٌ مِّنَ الرَّحْمَٰنِ الرَّحِيمِ ٤١ : ٢ : ٣
وَإِذَا بُشِّرَ أَحَدُهُم بِمَا ضَرَبَ لِلرَّحْمَٰنِ مَثَلًا ظَلَّ وَجْهُهُ مُسْوَدًّا وَهُوَ كَظِيمٌ ٤٣ : ١٧ : ٦
وَجَعَلُوا الْمَلَائِكَةَ الَّذِينَ هُمْ عِبَادُ الرَّحْمَٰنِ إِنَاثًا ٤٣ : ١٩ : ٦
وَقَالُوا لَوْ شَاءَ الرَّحْمَٰنُ مَا عَبَدْنَاهُم ٤٣ : ٢٠ : ٤
وَلَوْلَا أَن يَكُونَ النَّاسُ أُمَّةً وَاحِدَةً لَّجَعَلْنَا لِمَن يَكْفُرُ بِالرَّحْمَٰنِ لِبُيُوتِهِمْ سُقُفًا مِّن فِضَّةٍ وَمَعَارِجَ عَلَيْهَا يَظْهَرُونَ ٤٣ : ٣٣ : ١٠
وَمَن يَعْشُ عَن ذِكْرِ الرَّحْمَٰنِ نُقَيِّضْ لَهُ شَيْطَانًا فَهُوَ لَهُ قَرِينٌ ٤٣ : ٣٦ : ٥
وَاسْأَلْ مَنْ أَرْسَلْنَا مِن قَبْلِكَ مِن رُّسُلِنَا أَجَعَلْنَا مِن دُونِ الرَّحْمَٰنِ آلِهَةً يُعْبَدُونَ ٤٣ : ٤٥ : ١١
قُلْ إِن كَانَ لِلرَّحْمَٰنِ وَلَدٌ فَأَنَا أَوَّلُ الْعَابِدِينَ ٤٣ : ٨١ : ٤
مَّنْ خَشِيَ الرَّحْمَٰنَ بِالْغَيْبِ وَجَاءَ بِقَلْبٍ مُّنِيبٍ ٥٠ : ٣٣ : ٣
الرَّحْمَٰنُ ٥٥ : ١ : ١
هُوَ الرَّحْمَٰنُ الرَّحِيمُ ٥٩ : ٢٢ : ١٢
مَّا تَرَىٰ فِي خَلْقِ الرَّحْمَٰنِ مِن تَفَاوُتٍ ٦٧ : ٣ : ١٠
مَا يُمْسِكُهُنَّ إِلَّا الرَّحْمَٰنُ ٦٧ : ١٩ : ١١
أَمَّنْ هَٰذَا الَّذِي هُوَ جُندٌ لَّكُمْ يَنصُرُكُم مِّن دُونِ الرَّحْمَٰنِ ٦٧ : ٢٠ : ١٠
قُلْ هُوَ الرَّحْمَٰنُ آمَنَّا بِهِ وَعَلَيْهِ تَوَكَّلْنَا ٦٧ : ٢٩ : ٣
رَّبِّ السَّمَاوَاتِ وَالْأَرْضِ وَمَا بَيْنَهُمَا الرَّحْمَٰنِ ٧٨ : ٣٧ : ٦
لَّا يَتَكَلَّمُونَ إِلَّا مَنْ أَذِنَ لَهُ الرَّحْمَٰنُ وَقَالَ صَوَابًا ٧٨ : ٣٨ : ١٢

اسم بصيغة «رُحْم»

فَأَرَدْنَا أَن يُبْدِلَهُمَا رَبُّهُمَا خَيْرًا مِّنْهُ زَكَاةً وَأَقْرَبَ رُحْمًا ١٨ : ٨١ : ٩

اسم بصيغة «مَرْحَمَة»

ثُمَّ كَانَ مِنَ الَّذِينَ آمَنُوا وَتَوَاصَوْا بِالصَّبْرِ وَتَوَاصَوْا بِالْمَرْحَمَةِ ٩٠ : ١٧ : ٩

فعل من الثلاثي المجرد

ماض
مَّن يُصْرَفْ عَنْهُ يَوْمَئِذٍ فَقَدْ رَحِمَهُ ٦ : ١٦ : ٦
قَالَ لَا عَاصِمَ الْيَوْمَ مِنْ أَمْرِ اللَّهِ إِلَّا مَن رَّحِمَ ١١ : ٤٣ : ١٧
إِلَّا مَن رَّحِمَ رَبُّكَ ١١ : ١١٩ : ٣
إِنَّ النَّفْسَ لَأَمَّارَةٌ بِالسُّوءِ إِلَّا مَا رَحِمَ رَبِّي ١٢ : ٥٣ : ١٠
وَلَوْ رَحِمْنَاهُمْ وَكَشَفْنَا مَا بِهِم مِّن ضُرٍّ لَّلَجُّوا فِي طُغْيَانِهِمْ يَعْمَهُونَ ٢٣ : ٧٥ : ٢
وَمَن تَقِ السَّيِّئَاتِ يَوْمَئِذٍ فَقَدْ رَحِمْتَهُ ٤٠ : ٩ : ٨
إِلَّا مَن رَّحِمَ اللَّهُ ٤٤ : ٤٢ : ٣
قُلْ أَرَأَيْتُمْ إِنْ أَهْلَكَنِيَ اللَّهُ وَمَن مَّعِيَ أَوْ رَحِمَنَا فَمَن يُجِيرُ الْكَافِرِينَ مِنْ عَذَابٍ أَلِيمٍ ٦٧ : ٢٨ : ٩
مضارع
وَأَطِيعُوا اللَّهَ وَالرَّسُولَ لَعَلَّكُمْ تُرْحَمُونَ ٣ : ١٣٢ : ٥
وَهَٰذَا كِتَابٌ أَنزَلْنَاهُ مُبَارَكٌ فَاتَّبِعُوهُ وَاتَّقُوا لَعَلَّكُمْ تُرْحَمُونَ ٦ : ١٥٥ : ٨
قَالَا رَبَّنَا ظَلَمْنَا أَنفُسَنَا وَإِن لَّمْ تَغْفِرْ لَنَا وَتَرْحَمْنَا لَنَكُونَنَّ مِنَ الْخَاسِرِينَ ٧ : ٢٣ : ٩
أَوَعَجِبْتُمْ أَن جَاءَكُمْ ذِكْرٌ مِّن رَّبِّكُمْ عَلَىٰ رَجُلٍ مِّنكُمْ لِيُنذِرَكُمْ وَلِتَتَّقُوا وَلَعَلَّكُمْ تُرْحَمُونَ ٧ : ٦٣ : ١٣
وَلَمَّا سُقِطَ فِي أَيْدِيهِمْ وَرَأَوْا أَنَّهُمْ قَدْ ضَلُّوا قَالُوا لَئِن لَّمْ يَرْحَمْنَا رَبُّنَا وَيَغْفِرْ لَنَا لَنَكُونَنَّ مِنَ الْخَاسِرِينَ ٧ : ١٤٩ : ١٢
وَإِذَا قُرِئَ الْقُرْآنُ فَاسْتَمِعُوا لَهُ وَأَنصِتُوا لَعَلَّكُمْ تُرْحَمُونَ ٧ : ٢٠٤ : ٨
أُولَٰئِكَ سَيَرْحَمُهُمُ اللَّهُ ٩ : ٧١ : ١٩
وَإِلَّا تَغْفِرْ لِي وَتَرْحَمْنِي أَكُن مِّنَ الْخَاسِرِينَ ١١ : ٤٧ : ١٦
عَسَىٰ رَبُّكُمْ أَن يَرْحَمَكُمْ ١٧ : ٨ : ٤
إِن يَشَأْ يَرْحَمْكُمْ أَوْ إِن يَشَأْ يُعَذِّبْكُمْ ١٧ : ٥٤ : ٦
وَأَقِيمُوا الصَّلَاةَ وَآتُوا الزَّكَاةَ وَأَطِيعُوا الرَّسُولَ لَعَلَّكُمْ تُرْحَمُونَ ٢٤ : ٥٦ : ٨
لَوْلَا تَسْتَغْفِرُونَ اللَّهَ لَعَلَّكُمْ تُرْحَمُونَ ٢٧ : ٤٦ : ١٢
يُعَذِّبُ مَن يَشَاءُ وَيَرْحَمُ مَن يَشَاءُ ٢٩ : ٢١ : ٤
وَإِذَا قِيلَ لَهُمُ اتَّقُوا مَا بَيْنَ أَيْدِيكُمْ وَمَا خَلْفَكُمْ لَعَلَّكُمْ تُرْحَمُونَ ٣٦ : ٤٥ : ١١
وَاتَّقُوا اللَّهَ لَعَلَّكُمْ تُرْحَمُونَ ٤٩ : ١٠ : ١٠
أمر
وَاعْفُ عَنَّا وَاغْفِرْ لَنَا وَارْحَمْنَا ٢ : ٢٨٦ : ٤٣
أَنتَ وَلِيُّنَا فَاغْفِرْ لَنَا وَارْحَمْنَا ٧ : ١٥٥ : ٣٨
وَاخْفِضْ لَهُمَا جَنَاحَ الذُّلِّ مِنَ الرَّحْمَةِ وَقُل رَّبِّ ارْحَمْهُمَا كَمَا رَبَّيَانِي صَغِيرًا ١٧ : ٢٤ : ٩
إِنَّهُ كَانَ فَرِيقٌ مِّنْ عِبَادِي يَقُولُونَ رَبَّنَا آمَنَّا فَاغْفِرْ لَنَا وَارْحَمْنَا وَأَنتَ خَيْرُ الرَّاحِمِينَ ٢٣ : ١٠٩ : ١١
وَقُل رَّبِّ اغْفِرْ وَارْحَمْ وَأَنتَ خَيْرُ الرَّاحِمِينَ ٢٣ : ١١٨ : ٤
